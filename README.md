# 💧 LiquidPad

**A fluid, transparent notepad with stunning glass effects.**

LiquidPad is a minimalistic, beautiful text editor built with Python and Tkinter. It features adjustable transparency, multiple themes with glass morphism effects, and live writing statistics — all while being incredibly lightweight (~25MB RAM).

![LiquidPad Screenshot](assets/screenshots/glass-theme.png)

---

## ✨ Features

🪟 Window Transparency — Adjustable from 50% to 95% opacity
💎 Glass Morphism — Frosted glass effects on select themes
🌈 7 Beautiful Themes — Glass Dark, Glass Light, Dark, Light, Midnight, Forest, Ocean
📊 Live Statistics — Real-time word and character count
🔤 Font Control — 8pt to 24pt sizes with Ctrl+Scroll zoom
⌨️ Keyboard Shortcuts — Full suite for power users
💾 File Operations — New, Open, Save, Save As with all file types
⚡ Ultra Lightweight — ~25MB RAM, instant startup
🎯 Zero Dependencies — Pure Python standard library
🖼️ Custom Icon — Beautiful taskbar and window icon
💾 Session Recovery — Auto-saves every 30 seconds, restores unsaved work after crash
📄 Recent Files — Quick access to last 10 opened files
📏 Line Numbers — Synced gutter with current line highlighting
🖱️ Right-Click Menu — Cut, Copy, Paste, Select All, Undo, Redo, Clear Selection
🌐 Web Search — Right-click selected text to search Google instantly
📊 Selection Stats — Shows "Words: 5/45 selected" when text is highlighted
📁 File Type Detection — Auto-detects 35+ file types with icons (🐍 Python, 🌐 HTML, 🎨 CSS)
🔄 Session Memory — Remembers theme, opacity, window size & position on relaunch
🔍 Find & Replace — Full search with match case, whole word, highlight-all, and replace-all
🧹 Smart Cleanup — Auto-deletes recovery files when work is saved
📂 All File Types — Open dialog shows all files by default, 9 filter categories
🎨 Theme Line Numbers — Line numbers adapt colors to match every theme
